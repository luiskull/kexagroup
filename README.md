# Kexa Group

Sitio web de Kexa Group — página principal con aviso de "Coming Soon".

## Requisitos

- Node.js 18+
- npm

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en el navegador.

## Producción

```bash
npm run build
npm run preview
```

## Estructura

- `public/coming-soon.png` — imagen principal de la landing
- `src/App.jsx` — página principal
- `.env` — variables de entorno (ignorado por git)
